Primary Button:

```js
<Buttons textColor="text-white" bgColor="bg-main">
  Primary Button
</Buttons>
```

Disabled Button:

```js
<Buttons textColor="text-white" bgColor="bg-main" disabled>
  Disabled Button
</Buttons>
```

Outline Button:

```js
<Buttons textColor="text-link" border="border border-main">
  Outline Button
</Buttons>
```

Offer Button:

```js
<Buttons textColor="text-link" bgColor="bg-shades20">
  Offer Button
</Buttons>
```
